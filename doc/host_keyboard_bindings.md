### Functional buttons
- Volume up         - key [ `R` ] 
- Volume down 		- key [ `F` ]
- Functional left 	- key [ `Q` ]
- Functional right 	- key [ `E` ]
- Joystick up 		- key [ `W` ]
- Joystick down		- key [ `X` ]
- Joystick left 		- key [ `A` ]
- Joystick right		- key [ `D` ]
- Joystick enter		- key [ `S` ]
- Slider up			- key [ `V` ]
- Slider mid		- key [ `B` ]
- Slider down 		- key [ `N` ]
- Torch button 		- key [ `T` ]

### Numeric buttons
- Numeric 0		- key [ `0` ]
- Numeric 1		- key [ `1` ]
- Numeric 2		- key [ `2` ]
- Numeric 3		- key [ `3` ]
- Numeric 4		- key [ `4` ]
- Numeric 5		- key [ `5` ]
- Numeric 6		- key [ `6` ]
- Numeric 7		- key [ `7` ]
- Numeric 8		- key [ `8` ]
- Numeric 9		- key [ `9` ]
- Numeric Ast		- key [ `Z` ]
- Numeric Pnd		- key [ `C` ]

### Battery buttons
 - Connect/disconnect wall adapter		- key [ `P` ]
 - Decrease battery level		- key [ `[` ]
 - Increase battery level		- key [ `]` ]
