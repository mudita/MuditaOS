# libmtp - Media Transfer Protocol responder implementation

## requirements

Unit test requires ```cgreen-dev``` to be installed. Check Dockerfile or visit *cgreen-dev* webpage.


## build and test
```
make
```

To skip unit test just do:
```
make lib
```
